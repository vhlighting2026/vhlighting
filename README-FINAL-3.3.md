# VH Lighting Final 3.3

## Nâng cấp nhập CSV
- File CSV bắt buộc đủ 6 cột: sku, name, unit, light_attribute, price, stock.
- Xem trước đầy đủ 6 trường trước khi nhập.
- Kiểm tra từng dòng và chỉ rõ số dòng, lý do lỗi.
- Báo cáo sau nhập: tổng số dòng, thêm mới, cập nhật, bỏ qua và lỗi hệ thống.
- Sản phẩm trùng mã + tên + ánh sáng được cập nhật giá và tồn kho.
- Không sử dụng thư viện xlsx, phù hợp deploy Vercel.
